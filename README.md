## Getting Started

### Prerequisites

- Visual Studio or compatible C++ compiler
- raylib 5.5
- C++11 or later

### Building

1. Open `QuadTree2D.sln` in Visual Studio
2. Build the solution
3. Run the executable to visualize the quadtree

## Usage

The application provides an interactive visualization of a quadtree structure. Points can be inserted into the quadtree, and the spatial partitioning will be displayed in real-time.
